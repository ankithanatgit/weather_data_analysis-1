# Weather Data Analysis Report

## Objective
To analyze historical weather data and identify key patterns such as temperature variation, humidity trends, and rainfall behavior.

## Dataset Overview
The dataset contains daily records of:
- Temperature
- Humidity
- Rainfall
- Wind Speed

## Analysis Steps
1. Loaded dataset using Pandas.
2. Cleaned missing values.
3. Converted date columns to datetime format.
4. Performed exploratory data analysis.
5. Generated visualizations for trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
